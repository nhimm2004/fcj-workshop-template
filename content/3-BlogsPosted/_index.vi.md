Tại đây sẽ là phần liệt kê, giới thiệu các blogs mà các bạn đã đăng trên [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj). Ví dụ:

###  Blog 1 - Cách sử dụng AD FS user và Tableau để truy vấn dữ liệu một cách an toàn trong AWS Lake Formation

AWS Lake Formation hỗ trợ tích hợp với các nhà cung cấp danh tính (Identity Provider - IdP) như Microsoft Active Directory Federation Services (AD FS) thông qua SAML 2.0, cho phép người dùng sử dụng tài khoản Active Directory của doanh nghiệp để xác thực khi truy cập dữ liệu. Khi kết hợp với Tableau và Amazon Athena, người dùng có thể truy vấn dữ liệu trong Data Lake một cách an toàn mà không cần tạo IAM User hoặc sử dụng Access Key tĩnh.

###  Blog 2 - Xây dựng kiến trúc dữ liệu hiện đại đa Region và đáng tin cậy bằng AWS Glue và AWS Lake Formation
Xây dựng một Data Lake đa Region (Multi-Region Data Lake) trên AWS nhằm:
Đảm bảo tính sẵn sàng cao (High Availability).
Hỗ trợ khôi phục sau thảm họa (Disaster Recovery - DR).
Quản lý và chia sẻ dữ liệu tập trung.
Phân quyền chi tiết đến từng bảng, cột hoặc hàng.
Phục vụ phân tích dữ liệu theo thời gian thực và theo lô (batch).
